Método da Secante

UEPB, 06 de Outubro - 2016

Método baseado em aproximação linear, que vem pra diminuir as desvantagens que o método de Newton tem de calcular o $f’(x)$.

Para eliminar essa desvantagem, substituímos a derivada $f’(x)$ pela subtração entre $x_k$ e $x_{k-1}$

$f'(x_{k}) ≃ \frac{f(x_{k}) - f(x_{k-1})}{x_{k} - x_{k-1} }$

Onde $x_{k}$ e $x_{k-1}$ são duas aproximações para a raiz. Nesse caso a função de iteração fica:

$x_{k+1}=x_{k} - \frac{ \frac{ f(x_{k}) }{ f(x_{k}) - f(x_{k-1}) } } { x_{k}- x_{k-1} }$

Colocando sobre o mesmo denominador, temos a equação mais simples:

$x_{k+1} = \frac{ (x_{k-1} x f(x_{k}) - x_{k} x f(x_{k-1})}{x_{k} - x_{k-1} }$

Exemplo 1: Utilizando o método da Secante determine uma aproximação para a raiz da função $F$, com $e < 0.01$

$F(x)= \sqrt{5} - 4.e^{-x}$

Resolução:

Para facilitar o processo podemos utilizar o teorema de Bolzano para encontramos um intervalo que contenha a raiz. Aplicando valores em $F$ temos:

$x$012
$F(x)$ -5 -0.839 0.738
[+ -] ← Possui raiz

Agora já sabemos que entre $[ 1 , 2 ]$ possui pelo menos uma raiz, basta aplicarmos os valores na fórmula da secante. Assim temos:

$ X_0 = 1 → F(X_0)=-0.839 $

$ X_1 = 2 → F(X_1)=0.738 $

Utilizando os dados acima e aplicando na fórmula temos:

Primeira iteração

$X_2 =\frac{1*(0.738)-2*(-0.839)}{(0.738)-(-0.839)} → X_2 = 1.532 → F(x_2) = 0.157$

Segunda iteração

$X_3 = \frac{2*(0.157)- 1.532*(0.738)}{(0.157)-(0.738)} → X_3 = 1.405 → F(X_3) = 0.042 $

$ e=\left|\frac{1.405-1.532}{1.405}\right| = 0.09 $

Terceira iteração

$X_4={1.532*(-0.042)-(1.405)*(0.157)}{(-0.042)-(0.157)} → X_4 = 1.432 → F(X_4)=0.003$

$e = \left|\frac{1.432 - 1.405}{1.432}\right| = 0.02 $

Quarta iteração

$ X_5 = {1.405*(0.003)-(1.432)*(-0.042)}{(0.003)-(-0.042)} → X_5 = 1.4302 → F(X_5) = -0.0004 $

$ e= \left|\frac{1.4302 - 1.432}{1.4302}\right| = 0.001 $

Portanto $X5 = 1.4302$ é uma aproximação da raiz de $F$.

Exemplo 2: O volume $v$ de um líquido em um tanque esférico de raio $r$ está relacionado com a profundidade h do líquido da seguinte forma:


$v=\frac{\pi .h^2.(3r-h)}{3}$

Calcule, utilizando o método da secante, a profundidade $h$ em um tanque de raio $r = 1$ para um volume de $0.5$. Utilize para aproximação inicial o intervalo $[0.25, 0.5]$ e considere $E < 10^{-2}$.

Resolução:

O primeiro passo que devemos fazer é aplicar os valores dados no problema e isolarmos o $h$. Fazendo isso temos:

$0.5 = \frac{ \pi .h^2 * (3*1 - h)}{3} → 3 \pi .h^2 - \pi .h^3 = 1.5 → 3 \pi .h^2 - \pi h^3 - 1.5$

$F(h) → 3\pi h^2 - \pi h^3 - 1.5 = 0 $

Primeira iteração

$ H_2 = \frac{0.25*(-0.46) -0.5*(-0.96)}{(0.46)-(-0.96)} → H_2 = 0.419 → F(H_2) = -0.076$

Segunda iteração

$H_3 = \frac{0.5*(-0.076) -0.419*(0.46)}{(-0.076)-(0.46)} → H_3 = 0.43 → F(H_3) = -0.007$

$e=\left|\frac{0.43 - 0.419}{0.43}\right| = 0.026$

Terceira iteração

$H_4 = \frac{0.419*(-0.007) -0.43*(-0.076)}{(-0.007)-(-0.076)} → H_4 = 0.431 → F(H_4) = -0.0007$

$e=\left|\frac{0.431-0.43}{0.431}\right| = 0.002 $

Portanto H_4 = 0.431 é uma aproximação da raiz de $F$

Agora que encontramos nossa $F(h)$, podemos aplicar na fórmula da secante utilizando o intervalo dado no problema. Assim temos:

$H_0 = 0.25 → F(H_0) = -0.96$

$H_1 = 0.5 → F(H_0) = 0.46$

Método da Falsa Posição (Regula Falsi)

O método Regula Falsi (ou falsa posição) é uma extensão aprimorada do método da secante, no sentido de realizar menos iterações para se descobrir a raiz. Consiste no seguinte:

1. Determina-se os valores $[a,b]$ tal que $f(a) x f(b) < 0$, para que esse intervalo contenha a raiz;

2. Aplica-se então a mesma fórmula do método da secante, que é:

$X_{k+1} = \frac{X_{k-1}*F(X_k) - X_k*F(X_{k-1})}{F(X_k)- F(X_{k-1})}$

3. Eis então o ponto que diferencia o método Regula Falsi do método da secante: toda iteração $X_{k+1}$ deverá obedecer a regra: $f(X_{k+1}) * f( X_k) < 0$

Ou seja,

Consideremos um intervalo $[x_1,x_2]$ , onde $f(x_1) > 0 $ e $f(x_2) < 0$

  • Se o elemento $x_3$ descoberto resultar em $f(x_3) > 0$ , o elemento $x_4$ será calculado com o elemento $x_3$ e o elemento $x_2$

  • Se o elemento $x_3$ descoberto resultar em $f(x_3) < 0$ , o elemento $x_4$ será calculado com o elemento $x_3$ e o elemento $x_1$

Essa regra se aplica aos próximos elementos descobertos na fórmula $x_(k+1)$. Com isso, o número de iterações é significativamente reduzido, caso comparado ao método da secante.

Vale salientar que o cálculo do erro é o mesmo utilizado no método da secante, e serve para que, de acordo com a aproximação do erro, seja determinado a raiz do problema.

Resolução dos exemplos 1 e 2 utilizando o método da Falsa Posição (Regula Falsi)

Exemplo 1: Utilizando o método da Falsa Posição (ou Regula Falsi) determine uma aproximação para a raiz da função $F$, com $e < 0.01$.

Do exemplo resolvido pelo método da Secante, sabemos que entre $[ 1 , 2 ]$ há pelo menos uma raiz. Assim temos:

$X0 = 1 → F(X0)=-0.839$

$X1 = 2 → F(X1)=0.738$

Primeira iteração

A primeira iteração será idêntica ao método da secante visto anteriormente, teremos:

Segunda iteração

Como $F(x_0)*F(x_2) < 0 $, serão utilizados o $x_0$ e o $x_2$ para que seja determinado o $x_3$:

$x_0 = 1 → F(x_0) = -0.839$

$x_2 = 1.532 → F(x_2)= 0.157$

Teremos,

$x_3=\frac{x_0*F(x_2) - x_2*F(x_0)}{F(x_2) -F(x_0)} =$

$x_3 =\frac{1*0.157 - 1.532*(-0.839)}{0.157- (-0.839)}$

$x_3=1.448 ⇒F(x_3)= 0.028$

Cálculo do Erro:

$е = \left|\frac{x_3 - x_2}{x_3}\right| = \left|\frac{1.448 - 1.532}{1.448}\right| = 0.058$ . Maior do que 0.01, então vamos para a terceira iteração.

Terceira iteração

Utilizando a comparação explicada na iteração anterior, serão utilizados os seguintes parâmetros para a determinação do $x_4$:

$x_0 = 1 F(x_0) = -0.839$

$x_3 = 1.448 F(x_3)= 0.028$

$x_4=\frac {x_{0}*F(x_{3}) - x_{3}*F(x_{0})}{F(x_{3}) -F(x_{0})} =$

$x_4 =\frac { 1*0.028 - 1.448 * (-0.839)}{0.028- (-0.839)}$

$x_4= 1.434⇒F(x_4) = 0.0057$

Cálculo do Erro:

$е = \left|\frac{x_4 - x_3}{x_4}\right| = \left|\frac{1.434 - 1.448}{1.434}\right| =0.009$ . Erro menor do que $0.01$, então paramos por aqui e obtemos o resultado satisfatório para a raiz que é 1.434.
