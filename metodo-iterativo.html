Método do Ponto Fixo (Iterativo Linear)

Seja f(x) uma função contínua no intervalo [a,b], intervalo que contém uma raiz da equação f(x) = 0. O método do Ponto Fixo consiste em fazer com que esta equação f(x) = 0 se torne uma equação equivalente da forma x = (x).

Após determinar uma função x que satisfaça o teorema, deve-se determinar um elemento x0, ou seja, uma aproximação inicial. Com isso, gera-se uma sequência xk de aproximações que convirjam para uma raiz x, tal que:

  • x1= (x0)
  • x2= (x1)
  • x3= (x2)
  • xn= (xn-1)

O processo iterativo terá a convergência mais rápida quanto menor for o valor de ' (x) . Entretanto, se a declividade ' (x) for maior que 1 em valor absoluto, para todo x que pertença ao intervalo I divergirá. O método iterativo linear é mais conceitual do que eficiente em relação ao uso computacional. Além do mais, é necessário realizar o teste de convergência, onde | ' (x)| < 1.
